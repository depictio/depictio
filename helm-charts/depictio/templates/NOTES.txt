{{- if contains "ClusterIP" .Values.frontend.service.type }}
To access the Depictio application from inside the cluster:

  Export the service port:
  $ kubectl port-forward -n {{ .Values.namespace.name }} service/depictio-frontend 8080:{{ .Values.frontend.service.httpPort }}

  Then visit: http://localhost:8080
{{- else if contains "LoadBalancer" .Values.frontend.service.type }}
To access the Depictio application:

  NOTE: It may take a few minutes for the LoadBalancer IP to be available.
        You can watch the status by running:
        $ kubectl get svc -n {{ .Values.namespace.name }} -w depictio-frontend

  Once the external IP is available, the Depictio application will be accessible at:
  http://EXTERNAL-IP
{{- end }}

{{- if contains "ClusterIP" .Values.minio.service.type }}
To access the MinIO console from inside the cluster:

  Export the console port:
  $ kubectl port-forward -n {{ .Values.namespace.name }} service/minio 9001:{{ .Values.minio.service.consolePort }}

  Then visit: http://localhost:9001
  
  Login with:
    Username: {{ .Values.secrets.minioRootUser }}
    Password: {{ .Values.secrets.minioRootPassword }}
{{- end }}
